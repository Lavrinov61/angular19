# Smoke Test Checklist

## Online Flow
- [ ] В конфигураторе выбрать опции и нажать `Заказать`
- [ ] Открывается `PhotoUploadOverlay`
- [ ] В overlay выбрать тип документа
- [ ] Загрузить 3+ фото и нажать `Отправить`
- [ ] Чат открывается, фото отправлены, шаг `document_select` не появляется
- [ ] Появляется `order_confirmed` с кнопкой оплаты
- [ ] Оплата через CloudPayments проходит и показывается success-экран
- [ ] Кнопки `Вернуться в чат` и `Новый заказ` работают

## AI Safety Net
- [ ] Сообщение с raw JSON function call не показывается клиенту
- [ ] Если AI вернул `{"name":"select_document","arguments":{"type":"passport_rf"}}`, действие выполняется как интерактивный шаг
- [ ] Обычные AI-ответы продолжают содержать chips

## Upload UX
- [ ] Desktop: overlay открывается как popup
- [ ] Mobile: overlay открывается fullscreen
- [ ] При открытом overlay фон страницы не скроллится
- [ ] Закрытие работает по `Esc`, по backdrop и по кнопке `Закрыть`

## Cart/Sync
- [ ] Добавление в корзину из чата синхронизируется с UI корзины
- [ ] Изменение корзины в UI отражается в чат-потоке

## Navigation Regression
- [ ] Welcome-кнопки работают без dead-end переходов
- [ ] Кнопки `◀ Назад` и `В меню` работают в критичных шагах оформления

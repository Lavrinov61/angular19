Как использовать

  • Откройте этот файл в Chrome, выделите нужный шаблон
  • Скриншот: Ctrl+Shift+S (Firefox) или DevTools → Capture node screenshot
  • Замените текст, цены, даты прямо в HTML
  • Для фото-фона: добавьте background-image на блок .hero-bg или .split-left
  • Размеры: 4:3 = 1200×900, 1:1 = 720×720, 9:16 = 1080×1920
  • Stories (9:16): замените src в <img> на нужное фото из assets/static/promo/

08 Контакт

Как экспортировать 1080×1920

  • Откройте файл в Chrome, нажмите F12 (DevTools)
  • Кликните на нужную .story в Elements
  • Правый клик на ноде → Capture node screenshot
  • Для точных 1080×1920: в DevTools включите device toolbar (Ctrl+Shift+M), задайте 1080×1920, масштаб 100%
  • Меняйте текст, цены, даты прямо в HTML или через DevTools → Edit as HTML
  • Для фото-фонов: добавьте background-image: url(...) на .story-welcome, .svc-top и т.д.

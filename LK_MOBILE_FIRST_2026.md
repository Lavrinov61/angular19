## КЛЮЧЕВЫЕ ПРИНЦИПЫ

1. **Mobile-first = thumb-first** — все touch targets 44px+, все действия одной рукой
2. **Не сайт, а приложение** — haptic, звук, animations, offline, install prompt
3. **Чат = пульт управления** — заказ, оплата, согласование прямо в чате
4. **Не спамить, а помогать** — контекстный push, тихие часы, auto-suppress
5. **Конвертировать поводы** — фото-календарь, сезонные достижения, напоминания к датам
6. **Perceived performance** — skeleton > spinner, SWR > loading, optimistic UI
